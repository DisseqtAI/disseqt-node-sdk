# Examples

These examples are TypeScript source files intended to show the SDK surface.

- `validation.ts` - standard validator helper usage
- `composite-score.ts` - composite score evaluation
- `prompt-packs.ts` - prompt pack generation, run creation, and output validation
- `agentic-tracing.ts` - trace/span workflow instrumentation

Run them with a TypeScript runner such as `tsx` after replacing placeholder credentials.
